# 2. システム概要

## 2.1 システムの目的・背景

エス・エー・エスの営業担当者が顧客ごとの営業戦略情報を一元管理・共有するためのWebシステムです。  
従来はExcelで管理していた情報をWebアプリ化することで、複数担当者がリアルタイムに参照・更新できる環境を実現します。

**主な機能：**
- 顧客別営業戦略レコードの登録・編集・閲覧・削除
- 月次活動スケジュールのマトリクス管理
- 活動TODO（行動計画）の管理
- Excelへのエクスポート・インポート
- ユーザーアカウント管理（管理者機能）

## 2.6 Vercel URLルーティングルール

`vercel.json` で以下のルーティングが定義されています。

| URLパターン | 転送先 | 説明 |
|---|---|---|
| `/api/:path*` | `/api/server` | Serverless Functionへ転送 |
| `/(.*)`（それ以外） | `/index.html` | Vue SPAのエントリポイント |

Vue RouterによりSPA内でのページ遷移は `pushState` で処理されます。  
直接URLアクセスや画面リロード時も `index.html` にフォールバックするため、SPAが正しく動作します。
